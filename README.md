# MabuhAi

**MabuhAi** is a mental health and emotional wellness app designed to give users a safe, supportive, and calming digital space for reflection, connection, and self-care.

Built with **Tauri**, **React**, **TypeScript**, and **Vite**, the app combines an editorial, sanctuary-inspired UI with practical mental health tools that help users check in, express themselves, and find support when they need it most.

## ✨ Overview

MabuhAi is designed to support everyday well-being through guided emotional check-ins, AI-assisted conversations, private journaling, and safety-focused support features. The experience aims to feel warm, accessible, and non-judgmental.

## 🌿 Core Features

### 1. AI Chat Support
An intelligent chatbot that offers empathetic conversations, emotional check-ins, and guided support.

### 2. Daily Mood Checker
A simple daily tool for users to log how they feel and monitor emotional changes over time.

### 3. Mood Pattern Insights
Visual summaries and trends that help users better understand their moods, triggers, and habits.

### 4. Digital Thought Record
A private diary for capturing thoughts and emotions through **text, voice, or video entries**.

### 5. Mask-Off Mode
A safe, judgment-free space where users can express thoughts and feelings they may not be comfortable sharing elsewhere.

### 6. Predictive Self-Care Reminders
Smart reminders that encourage users to rest, hydrate, breathe, reflect, or take a break based on their activity and mood patterns.

### 7. Mental Health Tips & Affirmations
Pop-up encouragement, coping tips, and positive affirmations to support users throughout the day.

### 8. Activity Suggestions
Personalized wellness activities such as breathing exercises, journaling prompts, or calming routines based on the user’s current mood.

### 9. Nearby Psychologist Locator
GIS-based support to help users find the nearest psychologists, clinics, or mental health services.

### 10. Group Therapy / Community Support
A space for users to connect with others, join peer support groups, and feel less alone.

### 11. Gamification Engine
Users can earn **points, badges, and levels** by completing mood checks, journal entries, and other healthy activities.

### 12. Urgent Help Button
Quick access to emergency hotlines and crisis support resources when immediate help is needed.

### 13. AI Guardrails & Safety Monitoring
Built-in protective measures to detect harmful behavior, possible abuse, or crisis-related language and guide users to safer support options.

## 🛠️ Tech Stack

- **Frontend:** React + TypeScript + Vite
- **Desktop Runtime:** Tauri
- **Styling:** Tailwind CSS v4
- **UI Components:** shadcn-style primitives

## 🚀 Getting Started

### Prerequisites
- Node.js
- Rust
- Tauri development dependencies

### Run the app

```bash
npm install
npm run tauri dev
```

### Build the app

```bash
npm run build
```
